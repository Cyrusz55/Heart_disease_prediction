# Figures

Place exported charts from `book1.ipynb` in this directory.

Recommended formats:
- `PNG` for general use
- `SVG` for vector-quality figures
- `PDF` for publication workflows

Suggested naming:
- `eda_target_distribution.png`
- `eda_correlation_heatmap.png`
- `model_confusion_matrix.png`
- `roc_curves.png`

Reference these files from `README.md` or other project documentation using relative paths.
